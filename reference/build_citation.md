# Create a citation page for a lesson from a CITATION.CFF file

Create a citation page for a lesson from a CITATION.CFF file

## Usage

``` r
build_citation(pkg, quiet = FALSE)
```

## Arguments

- path:

  the path to the lesson. Defaults to current working directory

- out:

  the path to the citation page. When this is `NULL` (default) or
  `TRUE`, the output will be `site/built/citation.html`.
